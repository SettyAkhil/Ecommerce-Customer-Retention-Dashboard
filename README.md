# 🛍️ E-Commerce Customer Retention Dashboard

## 📊 Project Overview

This Power BI dashboard provides insights into customer retention behavior in an e-commerce platform.
It helps analyze repeat purchases, customer satisfaction, and factors influencing retention to support data-driven decision-making.
---

## 🗂 Dataset
- **File Used:** `ecommerce_retention_final.csv`
- The dataset includes:
  - Customer type (One-Time / Repeat)
  - Order and delivery details
  - Review scores
  - Payment types
  - Product categories
  - Customer locations

---

## 🌟 Key Insights
- Overall customer retention rate (1.39%).
- Comparison between one-time and repeat customers.
- Impact of delivery time on retention.
- Relationship between review scores and repeat purchases.
- Analysis of payment methods and product categories.
- Regional distribution of customers.

---

## 🖥️ Tools Used
- **Power BI Desktop** – Visualization and dashboard building.
- **Excel/CSV** – Data preparation.
- ***Python / Pandas*** – Data cleaning and preprocessing.
